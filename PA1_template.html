Devise a strategy for filling in all of the missing values in the dataset.

**** Strategy.
1 Calculate median steps per 5 minute interval using the ddply function from the plyr package. 2 Merge median values back to original data by interval 3 If steps value is missing (i.e. NA), replace steps with median1. 4 Use complete/imputed data set for calculations and plots.

s1 <- ddply(ds, .(interval), summarize, median1=median(steps, na.rm=TRUE))

dsi <- merge(ds, s1, by="interval", all.x=TRUE)

dsi$steps[is.na(dsi$steps)] <- dsi$median1[is.na(dsi$steps)]
